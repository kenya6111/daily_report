    ```python
    from django.contrib import admin
    from django.urls import path

    urlpatterns = [
        path('admin/', admin.site.urls),
    ]
    ```
    - path()の第一引数が受け取ったURLと合致させるものを書く。第二引数は合致した場合に呼び出す中身。第一引数を受け取り、第二引数という指示を出している
    この設定だと、http://localhost/adminを受けた時に発動する。
